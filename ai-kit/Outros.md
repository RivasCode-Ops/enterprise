Este arquivo é o **estacionamento** do ai-kit.

Use para:

- guardar prompts experimentais;
- anotar ideias de variações;
- registrar versões muito específicas que ainda não merecem arquivo próprio.

Quando algo daqui virar uso recorrente, mova para:

- um arquivo próprio (ex.: `X.md`), ou
- `Templates.md` (se for estrutura de prompt reutilizável).

## 2. Regras para este arquivo

- Não se preocupar com perfeição — este arquivo é lugar de teste.
- De tempos em tempos, revisar:
  - o que foi usado várias vezes → promover;
  - o que nunca foi usado → arquivar/apagar.

A função deste arquivo é evitar perder prompts úteis em conversas soltas, mas sem poluir o resto do kit.
