# Session Copy.md — Copy curta para sessão

Esta copy é uma versão curta da copy principal do `ai-kit`.

Use quando quiser iniciar uma sessão rápida com IA sem colar o prompt completo do `Copy.md`, mantendo a mesma lógica de prioridades entre projeto real, `dokit` e `ai-kit`.

---

## 1. Copy curta (geral)

````md
Atue neste projeto respeitando:

1. o contexto real do projeto;
2. o método universal do `dokit`;
3. o `ai-kit` apenas como apoio operacional.

Antes de implementar:
- leia a documentação viva do projeto (README, CONTEXT, HANDOFF, etc.);
- resuma o estado atual em poucas linhas;
- resuma a tarefa pedida;
- proponha um plano curto de execução.

Regras:
- não mude stack ou arquitetura sem sinalizar;
- não abra novas frentes fora do escopo atual;
- não trate templates ou prompts como se fossem o estado vivo do projeto;
- mantenha coerência com padrões existentes.

Ao finalizar:
- resuma o que foi feito;
- liste pendências e próximos passos;
- indique quais docs precisam ser atualizados (CONTEXT, HANDOFF, DECISIONS, etc.).

Objetivo da sessão:
[descreva aqui]
````

---

## 2. Versão extra-enxuta (one-liner estendido)

````md
Use o contexto real do projeto como fonte principal.
Use o `dokit` como método de trabalho.
Use o `ai-kit` apenas como apoio de prompt.

Antes de agir:
- confirme estado atual,
- resuma tarefa,
- proponha plano curto.

Durante:
- mantenha foco,
- não abra novas frentes,
- não altere arquitetura/stack sem sinalizar.

Ao final:
- resuma entregas, pendências, riscos e próximos passos.

Objetivo:
[descreva aqui]
````

---

## 3. Versão para retomada rápida

````md
Esta é uma sessão de retomada.

Antes de agir:
- leia o handoff e o contexto atual do projeto;
- resuma o ponto de retomada em 2–3 frases;
- proponha um plano curto apenas para concluir a pendência principal.

Regras:
- não abrir novas frentes;
- não refatorar grandes áreas neste momento;
- manter foco até encerrar a pendência principal.

Ao final:
- atualize o handoff com o novo estado;
- indique se o contexto geral mudou.

Objetivo da retomada:
[descreva aqui]
````

---

## 4. Como usar

- Cole a **Copy curta (geral)** ao iniciar uma sessão em qualquer ferramenta, quando não quiser usar o prompt completo.
- Use a **versão extra-enxuta** em prompts extremamente limitados (ex.: campo pequeno ou uso muito frequente).
- Use a **versão de retomada rápida** quando estiver continuando explicitamente de um handoff anterior.

Se precisar de mais detalhes, volte para:

- `ai-kit/Copy.md` para a versão completa;
- `dokit/START-SESSION.md` para a abertura universal com o método completo.

---

## 5. Gatilhos de atualização

Atualize este arquivo quando:

- a lógica principal do `Copy.md` mudar;
- a prioridade entre projeto real, `dokit` e `ai-kit` for alterada;
- surgirem novos padrões de uso que precisem de uma versão ainda mais curta ou ajustada.

A função desta copy é ser o **atalho prático** para começar sessões sem perder os princípios base.
