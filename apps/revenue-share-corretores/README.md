# revenue-share-corretores

Plataforma imobiliária da Riva's Tech onde corretores publicam imóveis e pagam apenas quando vendem.

## Objetivo

Unificar anúncio de imóveis, geração de leads, CRM e controle financeiro em um único sistema com modelo de revenue share.

## MVP

O foco inicial deste produto é entregar:

- cadastro e login de usuários;
- cadastro de corretores e compradores;
- publicação de imóveis;
- busca com filtros;
- formulário de contato e leads;
- painel básico do corretor;
- registro manual de venda e cálculo do split.

## Fora do MVP

Nesta fase, não entram:

- app mobile nativo;
- integração com portais imobiliários;
- chatbot avançado;
- split automático;
- IA avançada de precificação.

## Stack

- Backend: Laravel 11
- Frontend: Livewire + Alpine.js + Tailwind CSS
- Banco: PostgreSQL
- Cache/filas: Redis
